# Features

## Navigation System

### Keyboard Navigation
- **Left/Right Arrows**: Navigate between slides
- **Escape**: Return to slide overview (future enhancement)
- Smooth transitions between slides

### Progress Indicator
- Visual progress bar at top of screen
- Current slide number / total slides
- Click to jump to specific slides (future enhancement)

### Touch/Swipe Support
- Swipe left/right on mobile devices
- Touch-friendly navigation buttons

## Interactive Workflow Diagram

### Visual Representation
- Animated SVG/CSS diagram showing:
  - Plan phase
  - Implement phase
  - Verify phase
  - Review phase
- Cyclical flow visualization

### Interactivity
- Click on phases to expand details
- Hover effects showing related commands
- Animated transitions between states

## Command Showcase

### Command Cards
Each command displayed with:
- Command name (e.g., `/dev:feature`)
- Description of what it does
- Example usage with syntax highlighting
- Related agent information
- Tips for best results

### Command Categories
1. **Development** (`/dev:*`)
   - `/dev:feature` - Plan new features
   - `/dev:bug` - Debug issues
   - `/dev:chore` - Maintenance tasks
   - `/dev:refactor` - Improve code structure

2. **Verification** (`/verify:*`)
   - `/verify:types` - TypeScript checking
   - `/verify:lint` - Linting with Biome
   - `/verify:build` - Build verification
   - `/verify:loop` - Full verification cycle

3. **Review** (`/review:*`)
   - `/review:changes` - Review staged changes
   - `/review:perf` - Performance analysis

4. **Utilities** (`/utils:*`)
   - `/utils:commit` - Commit with message
   - `/utils:pr` - Create pull request
   - `/utils:cleanup` - Clean up code
